Modul zabezpečuje funkci tlačítek pro ovládání základních funkcí.

Dokumentace funkcí

void button_on ( char  button  )  [static]

Funkce je spouštěna při stisku tlačítka (náběžná hrana). Zde je místo pro implementaci případného filtru zákmitů. Pro každé konkrétní tlačítko je zde nabindována klávesa klávesnice. Je to provedeno voláním funkce keyb_in_buff(), která přidá do bufferu klávesnice kód libovolné klávesy.
